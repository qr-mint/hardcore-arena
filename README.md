# HardCore Arena

AI-powered hardcore game platform built for Telegram Mini Apps.

## 🚀 Overview

HardCore Arena is a real-time game platform where gameplay is combined with AI agents and blockchain infrastructure.

- 31 playable levels (live)
- Active users and working demo
- TON integration (payments + NFT minting)
- Built with React + Phaser

## 🤖 AI System (in progress)

The platform is evolving into a fully AI-driven system:

- Level Generator Agent — creates new hardcore levels
- Tester Agent — validates difficulty and playability
- Assistant Agent — interacts with players inside Telegram
